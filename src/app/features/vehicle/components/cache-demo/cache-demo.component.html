How to Test

  1. Click "Load Catalog" to fetch available chunks
  2. Select a year and make, then click "Load Chunk"
  3. Watch the progress bar as data downloads and compresses
  4. Check cache metrics to see storage usage
  5. Try loading the same chunk again (should be instant from cache)
  6. Click "Prefetch Popular" to download multiple chunks
  7. Open DevTools → Application → IndexedDB to see stored data
